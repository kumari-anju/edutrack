# 01 Project Setup

## Goal
Create the base Django project and folder structure for EduTrack.

## Tasks
- Create the Django project `edutrack` with `manage.py`.
- Add `requirements.txt` and `README.md` at the project root.
- Ensure the structure matches:
  - `edutrack/` project config
  - `apps/` for Django apps
  - `templates/` and `static/`

## Notes
- Use SQLite (default Django database).
- Keep the repo GitHub-ready.

When feature implemented successfully, move this file to `docs/completed/`.
