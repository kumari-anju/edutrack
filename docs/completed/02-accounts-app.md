# 02 Accounts App

## Goal
Create the `accounts` app for authentication and user profiles.

## Tasks
- Create Django app `accounts` inside `apps/`.
- Add login and logout views and routes.
- Add a user profile model with a `OneToOneField` to the Django user.
- Add templates for login and profile.

## Notes
- Use Django authentication best practices.

When feature implemented successfully, move this file to `docs/completed/`.
