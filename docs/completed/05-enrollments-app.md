# 05 Enrollments App

## Goal
Create the `enrollments` app to assign students to courses.

## Tasks
- Create Django app `enrollments` inside `apps/`.
- Add `Enrollment` model with proper `ForeignKey` relationships.
- Build views to assign students to courses.
- Add templates for enrollment creation and list views.
- Register URLs for enrollment routes.

When feature implemented successfully, move this file to `docs/completed/`.
